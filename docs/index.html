核心功能

每一瓦，尽在掌控

从菜单栏到全屏面板，PowerPulse 为你呈现 Mac 电源系统的完整面貌。

📊

实时功率监控

菜单栏显示当前功率（等宽数字无跳动），自动切换 ⚡🔌🔋 图标，一眼掌握充放电状态。

🌊

功率流可视化

动态流带展示电力走向：墙壁电源 → 系统 + 电池，粒子动画跟随功率实时变化。

🖥️

CPU / GPU / 内存

系统级监控面板，2秒刷新，5分钟趋势图表，逐核 CPU 使用率、GPU Metal 负载。

📈

充电历史

累计充电量（Wh）、平均/峰值功率、充电时长统计，完整记录每次充电会话。

🔮

毛玻璃外观

NSVisualEffectView 毛玻璃面板，自动适配深色/浅色模式，与 macOS 原生体验融为一体。

🔋

充电器识别

自动检测充电器型号、额定功率（V/A），在面板中清晰展示，不再猜测适配器规格。

状态色彩

颜色即信息

四种颜色，四种电源状态。一瞥之间，尽在掌握。

⚡
充电中
CHARGING
Mac 正在通过充电器充电，电池百分比持续上升。菜单栏图标显示 ⚡，功率流展示充电路径。
🔋
正常使用
ON BATTERY
未接入充电器，电池正常供电。电量处于健康范围内，菜单栏图标显示 🔋。
🔌
已充满 · 适配器供电
FULL · ADAPTER
电池 100% 满电，充电器仍在连接。系统直接使用外接电力，电池保持满电待命。
⚠️
低电量警告
LOW BATTERY
电池电量 ≤ 15%，未接入电源。需要尽快充电，这是最紧急的状态提示。
